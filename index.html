Not investment advice. Plutus trades imaginary money with simulated fills (5 bps fee + 5 bps slippage per side, applied to every bot). A one-month, ~40-decision run is a single noisy draw — most professional managers trail the index in any given year, and a random bot could plausibly win this window. This is an experiment in how an LLM decides, not evidence of edge. Rules were pre-registered before the first trade; the full audit trail is the git history of this repository.
